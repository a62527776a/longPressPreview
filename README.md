# flutter longPressPreview

用于任意组件长按并展示可拖动弹窗  
Long press and show draggable dialog of any widget.

可拖动弹窗以及多手势回调  
Draggable dialog and multi gesture callback provides.

上拉推入或者下拉关闭弹窗  
Drag up push or Drag down close dialog.
